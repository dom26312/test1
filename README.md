# Telnet Config Saver site

Статический сайт для GitHub Pages по PRD автоматического сохранения конфигурации плат по Telnet.

## Файлы

- `index.html` — основная страница.
- `styles.css` — адаптивный дизайн.
- `script.js` — интерактивная демонстрация логов.

## GitHub Pages

В репозитории откройте `Settings` → `Pages` и выберите:

- Source: `GitHub Actions`

Workflow `.github/workflows/pages.yml` опубликует статический сайт из корня репозитория.
После публикации сайт будет доступен по адресу GitHub Pages для репозитория.

## Безопасность

Не публикуйте реальные логины, пароли и IP-адреса оборудования в публичном репозитории. Храните их локально в конфиге или переменных окружения.
